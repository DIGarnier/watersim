# Archived documents

These reports predate the 2026-07-01 benchmarking pass. They were produced
against standalone benchmark programs that **reimplemented a simplified copy of
the physics** rather than driving the real `Physics` engine, so their numbers
don't describe this codebase; `OPTIMIZATIONS.md` also references code that has
since been rewritten. Kept for history. Current, reproducible results live in
`docs/benchmarks/` (harness: `benches/nbody.rs`).
